## Billing

### What payment methods do you accept?
- Credit/Debit Cards (Visa, Mastercard, Amex)
- PayPal
- Cryptocurrency (BTC, ETH, USDT)
- Account Balance

### Can I get a refund?
- **Within 24 hours**: Full refund, no questions asked
- **Within 7 days**: Prorated refund
- **After 7 days**: Contact support

### How does billing work?
Services are billed at the start of each billing cycle (monthly, quarterly, or yearly). You'll receive an invoice 3 days before the due date.

## Servers

### How long does it take to set up a server?
Most servers are deployed within 60 seconds of payment.

### Can I upgrade my server later?
Yes! You can upgrade RAM, storage, or CPU at any time. Changes take effect immediately.

### Do you offer DDoS protection?
Yes, all servers include enterprise-grade DDoS protection at no extra cost.

### Can I install custom mods/plugins?
Absolutely! You have full FTP access and can install any compatible mods or plugins.

## Technical

### What control panel do you use?
We use a custom-built control panel based on Pterodactyl, offering:
- File manager
- Console access
- Scheduled tasks
- Backup management
- Plugin installer

### Do you provide backups?
Yes! We offer:
- **Automatic**: Daily backups retained for 7 days
- **Manual**: Create backups anytime (stored for 30 days)

### Can I access my server via SSH?
SSH access is available on VPS and Dedicated server plans.
