# Installation

This guide covers how to set up your first server on OTH.

## Choosing a Server Type

OTH offers several hosting options:

| Type | Best For | Starting Price |
|------|----------|----------------|
| Game Servers | Minecraft, Rust, ARK | $5/month |
| Docker Containers | Web apps, APIs | $3/month |
| Dedicated Servers | High-performance needs | $50/month |

## Creating a Game Server

### Step 1: Select Your Game

1. Navigate to **Services** > **Create New**
2. Choose **Game Server**
3. Select your game from the list:
   - Minecraft (Java/Bedrock)
   - Rust
   - ARK: Survival Evolved
   - Valheim
   - And more...

### Step 2: Configure Your Server

```
Server Name: My Awesome Server
Region: US East
RAM: 4GB (recommended for 10-20 players)
Storage: 20GB SSD
```

### Step 3: Review and Deploy

1. Review your configuration
2. Select a billing cycle (monthly/quarterly/yearly)
3. Click **Deploy Server**

Your server will be ready in under 60 seconds!

## Connecting to Your Server

Once deployed, you'll receive:

- **Server IP**: `play.oth.gg:25565`
- **Control Panel Access**: Manage files, console, and settings
- **FTP Credentials**: For file uploads
